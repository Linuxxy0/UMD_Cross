# Contributing

欢迎基于这个项目继续扩展武侠世界与后续多世界内容。

## 分支建议
- `main`：稳定发布分支
- `feature/*`：新功能开发
- `fix/*`：问题修复
- `docs/*`：文档更新

## 提交建议
推荐使用以下前缀：
- `feat:` 新功能
- `fix:` 修复问题
- `docs:` 文档修改
- `refactor:` 重构
- `chore:` 杂项更新

## 开发前检查
1. 运行本地静态服务器确认页面可访问
2. 执行 `python3 scripts/validate_assets.py`
3. 确认新增 JSON 内容结构与现有配置兼容

## 内容扩展规范
- 事件 ID 保持唯一
- 新物品、技能、敌人都必须写入对应 JSON
- 新地图节点需要保证 `events` 或 `shopItems` 至少存在一种有效内容
- 新门派剧情优先写成配置，不要把剧情硬编码进场景
