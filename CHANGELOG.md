# Changelog

All notable changes to this project will be documented in this file.

## [1.2.0] - 2026-03-30

### Added
- 主界面升级为真正的页游 Hub：中央游戏区、右侧情报区、底部功能按钮栏
- 新增统一暗金属风格弹窗：个人信息、背包、商城、技能、地图
- 新增装备与物品悬停预览面板，支持在背包、商城与角色装备栏实时查看详情与属性变化

### Changed
- 世界地图页从纯探索页改为主界面承载页，快速操作统一收束到弹窗系统
- 存档键升级为 `transmigrator_wuxia_save_v120`
- 版本号提升为 `1.2.0`

## [1.1.0] - 2026-03-30

### Added
- 新增 `version.html` 作为仓库与 GitHub Pages 对外展示的版本页
- 新增 `assets/meta/version.json` 统一存放当前版本元信息
- 新增 README 发布页截图区与品牌文案区
- 新增仓库内置展示图：首屏、地图、门派装备、事件战斗

### Changed
- 统一仓库名建议为 `transmigrator-wuxia`
- 首页 header、首屏品牌文案与主菜单副标题改为公开发布口径
- 存档键升级为 `transmigrator_wuxia_save_v110`
- 版本号提升为 `1.1.0`

## [1.0.0] - 2026-03-30

### Added
- 正式仓库结构与 GitHub Pages 发布流程
- 武侠世界配置数据、地图、事件、敌人、技能与物品表
- 门派系统：天岳剑宗、玄衣楼、药王谷
- 装备系统：武器、护甲、饰品与商店购买
- 本地存档与导入导出
- 仓库维护文件：LICENSE、CONTRIBUTING、SECURITY、Issue/PR 模板
- 项目文档：部署、架构、内容配置、路线图

### Changed
- 首页与主菜单文案从演示性质改为正式发布项目表述
- 版本号提升为 1.0.0
- README 调整为正式仓库使用说明
